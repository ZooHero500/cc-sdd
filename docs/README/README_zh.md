一套支持 Claude Code、Cursor、Gemini CLI 和 Codex CLI 平台的综合规格驱动开发工具集。本项目在多个 AI 开发平台上复现了 Kiro IDE 的规格驱动开发工作流程。

**高 Kiro IDE 兼容性** — 无缝利用现有的 Kiro SDD 规格、工作流程和目录结构。

## 概述

本项目提供了一套工具集，用于在多个 AI 平台（Claude Code、Cursor、Gemini CLI、Codex CLI、GitHub Copilot、Qwen Code、Windsurf）上使用斜杠命令进行高效的规格驱动开发。通过在各开发阶段使用适当的命令，无论您偏好哪个平台，都可以实现系统化、高质量的开发流程。
